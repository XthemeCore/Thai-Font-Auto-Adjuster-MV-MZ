# Thai-Font-Auto-Adjuster-MV-MZ
Thai Font Automatic Adjuster plugin for RPG Maker MV/MZ

ปลั๊กอินนี้จะปรับตัวอักษรภาษาไทยให้เป็นไปตามยถากรรม 
การแสดงผลจะตรงกับที่แสดงในหน้าจอเริ่มเกมและเมนูต่างๆ

สามารถใช้ได้กับทั้ง RPG Maker MV และ MZ
